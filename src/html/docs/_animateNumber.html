Пример анимации чисел на сайте. Для примера анимируем число 15.

Для использования анимации, нужно число обернуть в любой html-тег и задать data-атрибут data-counter="XXX", где XXX - это число, которое нужно анимировать.

Так же можно задать data-атрибут data-duration="3000" с указанием времени анимации в мс. По умолчанию значение задано 2000 мс.

Еще есть возможно задать повторную анимацию числа, после того как пользователь еще раз при скроле сайта увидит это же число. Для этого нужно использовать data-атрибут data-repeat="true". По умолчанию значение задано false (откл).
