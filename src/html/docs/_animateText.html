Пример анимации залоголовка и текста на сайте. Для примера анимируем заголовок и ниже простой текст.

Для использования анимации заголовка, нужно его обернуть в любой html-тег и задать тегу класс .head-animate.

Еще есть возможно задать повторную анимацию заголовка, после того как пользователь еще раз при скроле сайта его снова. Для этого нужно использовать data-атрибут data-repeat="true". По умолчанию значение задано false, что означает что повтор аниамции отключен.

Так же есть еще параметр data-letter-delay="0.2", который отвечает за задержкe между буквами (по умолчанию 0.08s).

И есть второй параметр data-word-delay="0.4", который отвечает за задержкe между словами (по умолчанию 0.4s).

Чем значение в секундах выставлено больше, тем медленнее анимация.



Для использования анимации текста, нужно его обернуть в любой html-тег и задать тегу класс .text-animate.
